# 小狗效率屋 v1.0.0 使用说明

“小狗效率屋”是一款只在当前电脑保存数据的Windows待办与历史剪贴板工具，不需要登录，也不会云同步。

## 安装

1. 双击 `小狗效率屋-Setup-1.0.0-x64.exe`；
2. 按安装向导继续，默认安装不需要管理员权限；
3. 桌面快捷方式为可选项，默认不会创建；
4. 安装完成后可直接启动，开始菜单中也会出现“小狗效率屋”。

如果Windows显示“Windows已保护你的电脑”，这是因为本安装包暂未购买受信任代码签名证书。请确认文件来自本项目交付目录并核对SHA-256后再决定是否运行。

## 使用

- **待办：**输入内容后按Enter或点击添加；可编辑、完成、恢复或删除；
- **剪贴板：**程序运行期间会记录复制的文字和图片；每条内容都可重新复制；
- **搜索：**搜索框只匹配文字历史，图片不执行OCR；
- **管理：**文字和图片都可置顶、取消置顶和单条删除，文字还可以编辑；
- **窗口：**可拖动、缩放、折叠，关闭程序后停止剪贴板监听；
- **名称：**设置中可修改界面名称，安装名称仍保持“小狗效率屋”。

## 隐私提醒

- 数据目录：`%LocalAppData%\HMinus\DesktopSpike`；
- 复制的密码、验证码、密钥和敏感截图也可能被保存；
- 数据不会上传，但当前本机文件没有加密；
- 本版没有自动到期，请主动删除不需要的敏感记录；
- 退出软件只会停止监听，不会自动删除已经保存的历史。

## 卸载

卸载时会询问是否同时删除本机待办、剪贴板历史和设置。默认选择“否”以保留数据；只有明确选择“是”才会删除应用数据。重新安装后可以继续使用保留的数据。

## 当前限制

本版没有保存期限、自动清理、批量清空、开机启动、OCR、图片编辑、账号或云同步，仅支持Windows 10/11 x64。
