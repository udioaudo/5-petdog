# 小狗效率屋 v1.0.0 正式版发布与安装规范

## 1. 正式版本身份

- 产品名称：小狗效率屋；
- 正式版本：1.0.0；
- 支持系统：Windows 10 / Windows 11 x64；
- 内部程序文件：`HMinus.DesktopSpike.exe`；
- SQLite版本：v4；
- 本机数据目录：`%LocalAppData%\HMinus\DesktopSpike`；
- 默认安装目录：`%LocalAppData%\Programs\HMinus\DesktopEfficiency`。

软件内自定义名称只改变窗口和界面文字，不改变安装名称、安装目录、数据目录或升级身份。

## 2. 安装行为

- 使用Inno Setup 6.7.3构建中文安装包；
- 固定安装AppId为 `{5DFEAE73-9D75-4C6F-B8FA-2E89FAB075BF}`；
- 仅安装到当前Windows用户，不要求管理员权限；
- 始终创建开始菜单快捷方式；
- 桌面快捷方式为可选项，默认不勾选；
- 安装完成后的“立即运行”默认勾选；
- 不创建后台服务，不设置开机启动，不改变系统剪贴板设置；
- 覆盖安装使用同一AppId，只替换程序文件，不删除或迁移用户数据。

## 3. 卸载行为

- 卸载程序文件和快捷方式；
- 卸载时询问是否同时删除待办、剪贴板历史和设置，默认选择“否”；
- 只有用户明确选择“是”时，才删除 `%LocalAppData%\HMinus\DesktopSpike`；
- 删除路径在安装脚本中固定，不接受任意外部目录；
- 删除失败时完成程序卸载，并提示仍有本机数据残留；
- 默认卸载后重新安装，可以继续读取保留的数据。

## 4. v1.0.0冻结功能

- 待办添加、编辑、完成、恢复和删除；
- 完成历史按完成日期与时间展示；
- 文字和图片剪贴板历史本机保存；
- 文字搜索、逐条重新复制和文字编辑；
- 内容去重、置顶、取消置顶和单条删除；
- 自定义软件显示名称；
- 窗口拖动、缩放、位置记忆、折叠和始终置顶。

## 5. 已知限制

- 没有1天、2天、4天保存期限和自动清理；
- 没有批量清空或开机启动；
- 没有OCR、图片编辑、云同步、账号和跨设备功能；
- 本机SQLite和图片文件没有加密；
- 安装包未使用受信任代码签名，Windows可能显示SmartScreen提示；
- 正式测试环境为Windows 11 x64，Windows 10 x64未在当前电脑实际执行安装测试。

## 6. 可重复构建

```powershell
.\scripts\Build-Prototype.ps1
.\scripts\Test-Prototype.ps1
.\scripts\Build-Installer.ps1
.\scripts\Test-ReleaseUi.ps1
.\scripts\Test-Installer.ps1
```

Inno Setup编译器由 `scripts/Install-InnoSetup.ps1` 下载到项目 `.tools` 目录。脚本固定版本、文件长度和SHA-256，校验失败时停止构建。
